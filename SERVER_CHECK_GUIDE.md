# 리눅스 서버 스펙 확인 가이드

연구실 서버에서 딥러닝(Fine-Tuning)을 돌리기 전에, 하드웨어 사양을 확인하는 명령어들입니다. 터미널(SSH)에 접속해서 입력해 보세요.

## 1. GPU 확인 (가장 중요)
딥러닝 학습 속도를 결정하는 핵심 부품입니다.
```bash
nvidia-smi
```
*   **확인할 점**:
    *   **GPU Name**: 예) A100, V100, RTX 3090, RTX 4090, T4 등.
    *   **Memory-Usage**: 예) `0MiB / 24576MiB`. (총 메모리가 클수록 좋습니다. 최소 8GB 이상 권장, 24GB 이상이면 매우 좋음)
    *   **CUDA Version**: 우측 상단에 표시됨 (예: 11.8, 12.1). PyTorch 버전과 호환되어야 합니다.

## 2. CPU 확인
데이터 전처리 속도에 영향을 줍니다.
```bash
lscpu
```
*   **확인할 점**:
    *   **Model name**: CPU 모델명 (예: Intel Xeon, AMD EPYC).
    *   **CPU(s)**: 코어 개수. (많을수록 `num_workers`를 높게 설정 가능)

## 3. RAM (메모리) 확인
데이터를 한 번에 얼마나 올릴 수 있는지 결정합니다.
```bash
free -h
```
*   **확인할 점**:
    *   **Mem -> total**: 전체 메모리 용량. (32GB 이상 권장, 64GB 이상이면 넉넉함)
    *   **Mem -> available**: 현재 사용 가능한 여유 공간.

## 4. 디스크 용량 확인
데이터셋과 모델 체크포인트를 저장할 공간이 충분한지 확인합니다.
```bash
df -h
```
*   **확인할 점**:
    *   사용하려는 디렉토리(보통 `/home` 또는 `/data`)의 **Avail** (여유 공간)을 확인하세요.
    *   최소 50GB 이상의 여유 공간이 있는 것이 좋습니다.

## 5. OS 버전 확인
```bash
cat /etc/os-release
```
*   보통 Ubuntu 20.04 또는 22.04를 많이 사용합니다.

---
**팁**: `htop` 명령어를 입력하면 CPU와 메모리 사용량을 실시간으로 예쁘게 볼 수 있습니다. (설치되어 있다면)
