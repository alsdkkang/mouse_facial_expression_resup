# 프로젝트 현황 보고서: ReSup 모델 개발 및 평가
**날짜:** 2025년 11월 30일
**주제:** 모델 비교 분석 (Andre's Advanced vs. ReSup) 및 학습 진행 현황

## 1. 개요 (Executive Summary)
우리는 LDL(Label Distribution Learning), 시간적 집계(Temporal Aggregation), 그리고 이중 네트워크(Dual-Network) 아키텍처를 통합한 **ReSup (Reliable Noise Suppression) 모델**을 성공적으로 개발하고 배포했습니다.

**만성 스트레스(Chronic Stress) 데이터셋**에 대한 비교 평가 결과, 획기적인 성과를 확인했습니다:
- **Andre 모델 (기존)**: 스트레스 신호를 전혀 감지하지 못함 (**감지율 0%**).
- **ReSup 모델 (현재)**: 만성 스트레스 개체의 **19.5%**를 높은 확신도로 감지함.

현재 5-Fold 교차 검증(Cross-Validation) 과정은 **40% 완료**되었으며(Fold 0, 1 완료), 현재 로컬 환경에서 Fold 2 학습이 진행 중입니다.

---

## 2. 모델 비교: 기술적 아키텍처

ReSup 모델은 기존 아키텍처를 기반으로 하되, 라벨 노이즈를 처리하고 일반화 성능을 높이기 위한 핵심 메커니즘을 도입했습니다.

| 특징 | Andre 모델 (Advanced) | ReSup 모델 (Current) | 효과 |
| :--- | :--- | :--- | :--- |
| **핵심 구조** | 단일 네트워크 (ResNet34 + Transformer) | **이중 네트워크** (두 개의 독립된 ResNet34 + Transformer) | 학습 중 네트워크 간 상호 검증 가능. |
| **노이즈 처리** | 없음 (모든 라벨을 정답으로 간주) | **가중치 교환 (Weight Exchange)** | 네트워크 간 "의견 불일치"를 감지하여, 노이즈일 가능성이 높은 샘플의 가중치를 낮춤. |
| **일관성** | 없음 | **일관성 손실 (Consistency Loss)** | 두 네트워크가 안정적인 예측으로 수렴하도록 강제하여, 노이즈 과적합 방지. |
| **라벨링** | LDL (Soft Labels) | LDL (Soft Labels) | 둘 다 모호성을 포착하기 위해 Soft Label을 사용하나, ReSup은 이를 노이즈 억제 기술로 보완함. |

**핵심 통찰**: ReSup의 "Weak Supervision"은 외부 데이터를 추가하는 것이 아니라, 기존의 노이즈 섞인 라벨을 약한 신호(Weak Signal)로 취급하고 이중 네트워크의 불일치를 통해 필터링함으로써 달성됩니다.

---

## 3. 평가 결과: 만성 스트레스 데이터셋

급성 통증에 비해 감지하기 어려운 "숨겨진 고통"을 나타내는 **만성 스트레스 표정 영상** (41개)에 대해 두 모델을 평가했습니다.

### 정량적 결과

| 지표 | Andre 모델 | ReSup 모델 (Ensemble Fold 0+1) |
| :--- | :--- | :--- |
| **스트레스 감지** | **0 / 41 (0.0%)** | **8 / 41 (19.5%)** |
| **평균 스트레스 확률** | ~0.11 (매우 낮음) | **최대 0.84** (높은 확신도) |
| **감지 임계값** | 0.5를 넘지 못함 | 특정 코호트에서 임계값을 확실히 넘김 |

### 정성적 분석
- **Andre 모델**: 모든 개체에 대해 스트레스 확률을 거의 0에 가깝게 예측하는 "보수적" 성향을 보였습니다. 이는 급성 통증(Task 1)의 뚜렷한 신호에 과적합되어, 만성 스트레스의 미세한 신호에는 일반화되지 못한 것으로 보입니다.
- **ReSup 모델**: 미세한 특징에 대한 "민감도"를 보여주었습니다. 특히 **Cohort 1**에서 스트레스 사례를 집중적으로 식별했으며, 확률 점수는 **0.50에서 0.84** 사이로 나타났습니다. 이는 급성 통증의 "찡그림"과는 다른, 고통과 관련된 미묘한 표정 특징을 학습했음을 시사합니다.

## 4. 현재 진행 상황 및 향후 계획

### 학습 현황 (5-Fold 교차 검증)
통계적 신뢰성을 확보하기 위해 표준 Task 1 데이터셋에 대해 5-Fold 교차 검증을 진행 중입니다.

| Fold | 상태 | 결과 / 비고 |
| :--- | :--- | :--- |
| **Fold 0** | ✅ **완료** | 검증 정확도: **~99%** |
| **Fold 1** | ✅ **완료** | 검증 정확도: **~85%** |
| **Fold 2** | 🔄 **진행 중** | 로컬(MacBook Pro)에서 학습 시작됨 |
| **Fold 3** | ⏳ 대기 중 | Fold 2 완료 후 자동 시작 예정 |
| **Fold 4** | ⏳ 대기 중 | Fold 3 완료 후 자동 시작 예정 |

### 즉각적인 실행 계획
1.  **Fold 2-4 학습 모니터링**: 로컬 학습이 성공적으로 완료되는지 확인 (예상 소요 시간: 30-40시간).
2.  **최종 앙상블 평가**: 5개 Fold가 모두 완료되면, 5개 모델을 모두 사용하여 다시 앙상블 평가를 수행. 이를 통해 만성 스트레스 데이터셋에 대한 감지율과 신뢰도가 더욱 향상될 것으로 기대됨.
