# 프로젝트 현황 보고서: 만성 스트레스(Chronic Stress) 모델 개발
**작성일:** 2025년 12월 4일
**작성자:** Mina Kang

## 1. 프로젝트 개요
본 프로젝트의 목표는 기존에 급성 스트레스(Acute Stress, LPS) 데이터로 학습된 **ReSup 모델**이 **만성 스트레스(Chronic Stress)** 상황에서도 스트레스를 감지할 수 있는지 검증하고, 이를 위한 전용 모델을 개발하는 것입니다.

---

## 2. 1단계: Zero-Shot 평가 (초기 분석)
기존 모델(Acute Stress 학습)을 수정 없이 만성 스트레스 데이터셋(41개 비디오)에 바로 적용하여 성능을 평가했습니다.

### 주요 결과
*   **정확도(Accuracy)**: **45.0%** (무작위 추측보다 낮음)
*   **정밀도(Precision)**: 44.4%
*   **재현율(Recall)**: 19.0%

### 핵심 발견: Cohort 1의 배치 효과 (Batch Effect)
데이터를 분석한 결과, 모델이 **Cohort 1** 그룹에 속한 모든 쥐(9마리)를 실제 상태(Control/Stress)와 무관하게 **100% "Stress"로 예측**하는 현상이 발견되었습니다.
*   **원인 분석**: Cohort 1의 영상 촬영 환경(조명, 각도 등)이 기존 학습 데이터(Acute Stress)의 "Stress" 클래스와 시각적으로 유사하여 모델이 착각한 것으로 보입니다.
*   **결론**: 급성 스트레스 모델을 그대로 가져다 쓰는 것(Transfer Learning)은 한계가 있으며, 만성 스트레스 데이터에 맞는 **재학습(Fine-Tuning)**이 필수적입니다.

---

## 3. 2단계: 메타데이터 정립 및 전략 수립
연구팀(Brenna, Andre)과의 논의를 통해 데이터 처리 기준을 확립했습니다.

### 데이터 필터링 및 라벨링
1.  **제외 대상 (Excluded)**: 메타데이터에서 Grey/Yellow로 표시된 동물들은 실험 제외 개체이거나 이형접합자(Het)이므로 분석에서 **제외**했습니다.
2.  **KO 그룹 (Knockout)**: 유전자 변형 쥐(KO)의 경우, "WT Stress보다 더 심한 스트레스를 받았을 것"이라는 전문가 의견에 따라 **Stress (Class 1)**로 라벨링하여 학습에 포함시켰습니다.
    *   **전략**: 데이터 부족 문제를 해결하기 위해 **WT와 KO를 통합하여 학습(Binary Classification)**하되, 결과 분석 시에는 그룹별 성능을 따로 확인하기로 했습니다.

---

## 4. 3단계: Fine-Tuning (현재 진행 중)
위의 분석을 바탕으로, 만성 스트레스 데이터셋에 특화된 모델을 만들기 위해 **Fine-Tuning**을 진행하고 있습니다.

### 실행 계획
*   **환경**: Google Colab (A100/V100 GPU 활용)
*   **방법**: 5-Fold Cross-Validation (동물 ID 기준 분할)
*   **목표**:
    1.  Cohort 1에 대한 편향(Batch Effect) 제거.
    2.  만성 스트레스 특유의 미세한 표정 변화 학습.

---

3.  **최종 보고서 업데이트**: 모든 결과를 종합하여 최종 리포트를 완성합니다.

**요약**: 초기 모델 적용 시 **환경적 요인(Batch Effect)**으로 인한 성능 저하를 확인했으며, 이를 해결하기 위해 정확한 메타데이터를 기반으로 **Fine-Tuning**을 진행 중입니다.
